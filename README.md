# Kubernetes Ingress Dashboard

A real-time monitoring and navigation tool for Kubernetes ingresses.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Real-time Kubernetes ingress monitoring
- Search and filter functionality
- Light/dark/system theme support
- Attractive card-based visualization of ingresses
- Secure Kubernetes API authentication

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- shadcn/ui components
- Kubernetes JavaScript client
