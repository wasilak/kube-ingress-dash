# Requirements: Namespace Filter UI

## User Stories (EARS)
- When viewing the dashboard, users shall be able to filter ingresses by namespace using a selector next to the theme switcher.
- The namespace filter shall support selecting "All" and multiple namespaces at once.
- The UI shall update the displayed ingresses based on the selected namespaces.
- The filter shall be intuitive and accessible.

## Acceptance Criteria
- Namespace filter is present and functional.
- "All" and multiselect options work as expected.
- UI updates ingresses based on filter.
